    單元一：溫度計 — 一階物理系統

    從能量平衡推導水銀溫度計的一階轉移函數。

    溫度計問題

    考慮將一支水銀溫度計突然浸入溫度為 $T$ 的液體中。溫度計球部具有以下參數：

    • $A$ = 球部表面積 (m²)
    • $h$ = 薄膜熱傳係數 (W/m²·K)
    • $C_m$ = 球部中水銀的熱容量 (J/K)
    • $T$ = 周圍液體溫度 (°C)
    • $T_m$ = 水銀溫度（溫度計讀數）(°C)
    🔑
    目標：找出溫度計讀數 $T_m(t)$ 如何響應周圍溫度 $T(t)$ 的變化。

    圖示：浸入液體中的水銀溫度計（來源：教科書 & PPT 投影片第 2 頁）

    逐步推導

    第一步：能量平衡

    水銀的能量儲存速率等於從周圍液體的熱傳遞速率：

    $$C_m \frac{dT_m}{dt} = hA(T - T_m)$$

    此式表示：（熱容量）×（溫度變化率）=（對流熱傳遞速率）。

    第二步：穩態條件

    在穩態下，$\frac{dT_{m,s}}{dt} = 0$，因此：

    $$hA(T_s - T_{m,s}) = 0 \implies T_{m,s} = T_s$$

    穩態時溫度計讀數等於實際液體溫度。

    第三步：引入偏差變數

    定義偏差變數（偏離穩態的量）：

    $$x(t) = T(t) - T_s, \quad y(t) = T_m(t) - T_{m,s}$$

    將原始方程減去穩態方程：

    $$C_m \frac{dy}{dt} = hA(x - y)$$

    第四步：整理為標準形式

    兩邊除以 $hA$：

    $$\frac{C_m}{hA}\frac{dy}{dt} + y = x$$

    定義時間常數：$\tau = \dfrac{C_m}{hA}$，則：

    $$\boxed{\tau \frac{dy}{dt} + y = K_p \, x}$$

    其中溫度計的 $K_p = 1$（無因次增益）。

    第五步：取拉普拉斯轉換

    在零初始條件 $y(0) = 0$ 下：

    $$\tau s Y(s) + Y(s) = K_p X(s)$$ $$\boxed{G(s) = \frac{Y(s)}{X(s)} = \frac{K_p}{\tau s + 1}}$$

    這就是標準一階轉移函數。

    💡
    $\tau$ 的物理意義：$\tau = C_m / (hA)$。大的熱容量 $C_m$ 或小的熱傳係數 $h$ 意味著溫度計響應緩慢（大 $\tau$）。反之，高 $h$ 或小 $C_m$ 則響應迅速。

    一般一階系統

    任何可用一階 ODE 描述的物理系統都可以寫成：

    $$G(s) = \frac{Y(s)}{X(s)} = \frac{K_p}{\tau s + 1}$$

    其中：

    • $K_p$ = 穩態增益（穩態時輸出與輸入之比）
    • $\tau$ = 時間常數（表徵響應速度，時間單位）
    • $X(s)$ = 輸入的拉普拉斯轉換（可以是階躍、脈衝、斜坡、正弦等）

    為什麼要學三種標準輸入？

    一階系統有慣性 — 它無法瞬間跟上輸入的變化。為了全面理解這種動態行為，我們用三種典型輸入進行測試：

    輸入類型測試目的
    階梯（Step）系統多快能到達新穩態？
    斜坡（Ramp）系統能否持續追上不斷變動的輸入？
    正弦（Sinusoidal）系統對不同頻率的振盪如何響應？

    這三種測試共同揭示了系統的完整動態特性 — 速度、追隨能力和頻率濾波性質。

    單元二：一階系統的階躍響應

    分析一階系統對階躍變化的響應。

    階躍輸入定義

    階躍輸入，幅度為 $A$：

    $$x(t) = \begin{cases} 0 & t < 0 \\ A & t \geq 0 \end{cases} \quad \Longrightarrow \quad X(s) = \frac{A}{s}$$

    逐步推導：階躍響應

    第一步：代入轉移函數

    $$Y(s) = G(s) \cdot X(s) = \frac{K_p}{\tau s + 1} \cdot \frac{A}{s}$$

    第二步：部分分式展開

    $$Y(s) = \frac{K_p A}{s(\tau s + 1)} = K_p A \left(\frac{1}{s} - \frac{\tau}{\tau s + 1}\right) = K_p A \left(\frac{1}{s} - \frac{1}{s + 1/\tau}\right)$$

    第三步：拉普拉斯反轉換

    $$\boxed{Y(t) = K_p A \left(1 - e^{-t/\tau}\right)}$$

    這就是經典的一階階躍響應。

    階躍響應的關鍵特性

    時間$Y(t) / (K_p A)$最終值百分比
    $t = \tau$$1 - e^{-1} = 0.632$63.2%
    $t = 2\tau$$1 - e^{-2} = 0.865$86.5%
    $t = 3\tau$$1 - e^{-3} = 0.950$95.0%
    $t = 4\tau$$1 - e^{-4} = 0.982$98.2%
    $t = 5\tau$$1 - e^{-5} = 0.993$99.3%
    🔑
    經驗法則：經過 $5\tau$ 後，響應已達到最終值 $K_p A$ 的 99.3%。初始斜率為 $K_p A / \tau$，意即在 $t=0$ 處的切線會在 $t = \tau$ 時到達最終值。

    初始斜率與切線幾何意義

    微分：由 $Y(t)=K_pA(1-e^{-t/\tau})$ 對時間微分：

    $$\frac{dY}{dt} = \frac{K_p A}{\tau}\,e^{-t/\tau}$$

    系統在一開始上升最快，之後逐漸變慢。

    $t=0$ 時的初始斜率：

    $$\left.\frac{dY}{dt}\right|_{t=0} = \frac{K_p A}{\tau}$$

    原點處切線方程式：$Y_{\text{tangent}} = \dfrac{K_p A}{\tau}\,t$

    $\tau$ 的幾何意義：令切線等於最終值：

    $$\frac{K_p A}{\tau}\,t = K_p A \quad\Longrightarrow\quad t = \tau$$
    🔑
    重要幾何性質：在 $t=0$ 所畫出的初始切線，會在 $t=\tau$ 時碰到最終值 $K_pA$。但真正的響應曲線在 $t=\tau$ 時只到達最終值的 63.2% — 這兩件事不能混淆！

    安定時間（Settling Time）

    安定時間取決於誤差帶的定義：

    誤差帶安定時間響應水平
    2%$\approx 4\tau$98.2%
    1%$\approx 5\tau$99.3%
    ⚠
    注意：說「安定時間 $= 5\tau$」是常見的工程近似（對應 1% 誤差帶）。嚴格來說，安定時間取決於所採用的誤差標準。

    常見誤解

    #誤解正確說法
    1$t=\tau$ 時，系統已到達最終值。$t=\tau$ 時，響應僅為最終值的 63.2%。
    2$t=\tau$ 時，響應曲線與初始切線交於 $K_pA$。初始切線延伸才會在 $t=\tau$ 碰到 $K_pA$；真實曲線只到 63.2%。
    3$5\tau$ 是所有系統的精確安定時間。$5\tau$ 是一階系統 1% 誤差帶的近似值。
    4響應為 $y(t)=A(1-e^{-t/\tau})$。這僅適用於 $K_p=1$。一般式：$y(t)=K_pA(1-e^{-t/\tau})$。
    5$\tau$ 是系統達到穩態所需的時間。$\tau$ 是特徵時間。$t=\tau$ 時只完成 63.2%，需約 $5\tau$ 才接近穩態。

    物理解釋：階梯響應

    💡
    核心問題：如果輸入突然改變，系統要花多久才能跟上？

    施加階梯輸入後，輸出不會瞬間跳到最終值，而是一開始上升最快，然後逐漸變慢，漸近接近 $K_pA$。

    • $K_p$ 決定系統停在哪裡（最終值 $= K_pA$）
    • $\tau$ 決定系統多快到達
    • $\tau$ 越大 → 慣性越大 → 響應越慢 → 到穩態所需時間越長

    單元三：脈衝響應與斜坡響應

    另外兩種基本輸入：單位脈衝與斜坡函數。

    脈衝響應

    面積為 $A$ 的單位脈衝（Dirac delta 函數）：

    $$x(t) = A\,\delta(t) \quad \Longrightarrow \quad X(s) = A$$

    推導脈衝響應

    第一步：代入轉移函數

    $$Y(s) = \frac{K_p}{\tau s + 1} \cdot A = \frac{K_p A}{\tau s + 1}$$

    第二步：改寫為標準反轉換形式

    $$Y(s) = \frac{K_p A}{\tau} \cdot \frac{1}{s + 1/\tau}$$

    第三步：拉普拉斯反轉換

    $$\boxed{Y(t) = \frac{K_p A}{\tau}\,e^{-t/\tau}}$$

    脈衝響應為純指數衰減，從 $K_p A / \tau$ 衰減至零。

    🔑
    關鍵觀念：脈衝響應是階躍響應的微分。此外，脈衝響應曲線下的面積等於 $K_p A$（與階躍響應的最終值相同）。

    斜坡響應

    斜率為 $a$ 的斜坡輸入：

    $$x(t) = at \quad \Longrightarrow \quad X(s) = \frac{a}{s^2}$$

    推導斜坡響應

    第一步：代入轉移函數

    $$Y(s) = \frac{K_p}{\tau s + 1} \cdot \frac{a}{s^2} = \frac{K_p\,a}{s^2(\tau s + 1)}$$

    第二步：部分分式展開

    $$Y(s) = K_p a \left(\frac{1}{s^2} - \frac{\tau}{s} + \frac{\tau^2}{\tau s + 1}\right)$$

    第三步：拉普拉斯反轉換

    $$\boxed{Y(t) = K_p a \left(t - \tau + \tau\,e^{-t/\tau}\right)}$$

    第四步：穩態分析（大 $t$）

    當 $t \to \infty$ 時，$e^{-t/\tau} \to 0$，因此：

    $$Y(t) \approx K_p a (t - \tau) = K_p a\,t - K_p a\,\tau$$

    輸出最終以相同斜率 $K_p a$ 跟隨斜坡，但有 $\tau$ 的時間滯後（輸出永遠落後輸入 $\tau$ 個時間單位），並有 $-K_p a \tau$ 的穩態偏移。

    ⚠
    重要：對於斜坡輸入，一階系統永遠無法追上。始終存在 $\tau$ 的永久時間滯後和 $K_p a \tau$ 的永久偏移。減小 $\tau$ 可以同時減少滯後和偏移。

    物理解釋：斜坡響應

    💡
    核心問題：當輸入持續上升時，系統能不能一路跟上？

    與階梯（跳到固定值後停住）不同，斜坡輸入一直線性上升。系統不只要「會動」，還必須具備「持續追隨」的能力。

    • 輸出最終變成與輸入相同斜率 $K_pa$ 的直線
    • $K_p = 1$ 時：輸出是輸入向右延遲 $\tau$ 的形式
    • 在同一時間點，輸入與輸出的垂直差距約為 $K_pa\tau$
    • $\tau$ 越大 → 追隨越差 → 滯後和偏移越大

    圖形判讀：斜坡響應

    觀察下方互動圖表時注意：

    • 綠色虛線 = 輸入斜坡 $x(t) = bt$ — 從原點出發的直線
    • 藍色實線 = 輸出 $Y(t)$ — 起初落後，之後趨於平行
    • 暫態階段中，藍線逐漸追上相同斜率
    • 暫態消失後，注意時間滯後（水平間距 $= \tau$）和偏移（垂直間距 $= K_pb\tau$）
    • 增大 $\tau$ → 水平和垂直間距都變大
    • $K_p = 1$ 時：輸出線與輸入平行，僅有簡單的 $\tau$ 位移
    🔑
    關鍵結論：斜坡響應告訴你「系統能否持續追上變動中的輸入？」一階系統可以匹配斜率但永遠消除不了滯後。

    常見考點：斜坡響應

    主題關鍵事實
    斜坡輸入$x(t) = bt$，$X(s) = b/s^2$
    響應公式$y(t) = K_pb\left(t - \tau + \tau\,e^{-t/\tau}\right)$
    穩態近似$y(t) \approx K_pb(t - \tau)$（大 $t$ 時）
    永久時間滯後$\tau$
    永久偏移$K_pb\tau$
    $K_p=1$ 特例$y(t) \approx b(t-\tau)$：輸出 = 輸入向右平移 $\tau$

    單元四：正弦響應與頻率分析

    一階系統對正弦波輸入的響應 — 頻率響應分析的基礎。

    正弦輸入

    施加正弦輸入：

    $$x(t) = A\sin(\omega t) \quad \Longrightarrow \quad X(s) = \frac{A\omega}{s^2 + \omega^2}$$

    推導正弦響應

    第一步：s 域中的輸出

    $$Y(s) = \frac{K_p}{\tau s + 1} \cdot \frac{A\omega}{s^2 + \omega^2}$$

    第二步：部分分式與反轉換

    經過部分分式展開和拉普拉斯反轉換，解包含兩部分：

    $$Y(t) = \underbrace{\frac{K_p A \omega \tau}{1+\omega^2\tau^2}\,e^{-t/\tau}}_{\text{暫態（衰減消失）}} + \underbrace{\frac{K_p A}{\sqrt{1+\omega^2\tau^2}}\sin(\omega t + \phi)}_{\text{穩態正弦}}$$

    其中 $\phi = -\arctan(\omega\tau)$。

    第三步：穩態正弦響應

    暫態衰減後（當 $t \gg \tau$）：

    $$\boxed{Y_{ss}(t) = \frac{K_p A}{\sqrt{1 + \omega^2 \tau^2}} \sin(\omega t + \phi)}$$ $$\text{其中} \quad \phi = -\arctan(\omega \tau)$$

    頻率響應關鍵參數

    🔑
    振幅比 (AR)： $$AR = \frac{\text{輸出振幅}}{\text{輸入振幅}} = \frac{K_p}{\sqrt{1 + \omega^2\tau^2}}$$ 相位滯後 ($\phi$)： $$\phi = -\arctan(\omega\tau)$$
    $\omega\tau$$AR/K_p$相位滯後 $\phi$
    01.000$0°$
    0.50.894$-26.6°$
    1.00.707$-45.0°$
    2.00.447$-63.4°$
    5.00.196$-78.7°$
    $\infty$0$-90°$
    💡
    物理解釋：
    • 輸出始終是與輸入相同頻率的正弦波。
    • 振幅被衰減 — 頻率越高，衰減越大。
    • 輸出滯後於輸入 — 滯後隨頻率增加，最大為 $90°$。
    • 當 $\omega\tau = 1$ 時：振幅降至 $70.7\%$（$-3$ dB），相位滯後 $45°$。

    波德圖概念簡介

    波德圖（Bode Plot）是視覺化頻率響應的標準工具：

    • 振幅圖：$20\log_{10}(AR)$（dB）vs. $\log_{10}(\omega)$
    • 相位圖：$\phi$（度）vs. $\log_{10}(\omega)$

    對於一階系統 $G(j\omega) = \dfrac{K_p}{1 + j\omega\tau}$：

    • 低頻（$\omega \ll 1/\tau$）：振幅 $\approx 20\log_{10}(K_p)$ dB，相位 $\approx 0°$
    • 轉角頻率（$\omega = 1/\tau$）：振幅下降 $3$ dB，相位 $= -45°$
    • 高頻（$\omega \gg 1/\tau$）：振幅以 $-20$ dB/decade 下降，相位 $\to -90°$
    ⚠
    轉角頻率：$\omega_c = 1/\tau$ 是波德圖的「轉折點」。一階系統相當於低通濾波器 — 通過低頻信號，衰減高頻信號。

    圖形判讀：正弦響應

    觀察上方互動圖表時注意：

    • 綠色虛線 = 輸入 $x(t) = A\sin(\omega t)$ — 純正弦波
    • 藍色實線 = 輸出 $Y(t)$ — 起初有暫態，之後穩定為正弦波
    • 輸出仍是正弦波，但不是原封不動照抄輸入
    • 振幅變小 — 系統抑制了波動
    • 波峰晚到 — 系統引入了相位滯後
    • 增大 $\omega$ → 振幅更小、相位滯後更大
    • 增大 $\tau$ → 同樣效果：更大衰減、更多滯後
    🔑
    關鍵結論：正弦響應告訴你「系統如何處理振盪？」一階系統是低通濾波器 — 通過低頻信號，衰減高頻信號。

    常見考點：正弦響應

    主題關鍵事實
    正弦輸入$x(t) = A\sin(\omega t)$
    穩態輸出振幅$K_pA \,/\, \sqrt{1 + \omega^2\tau^2}$
    振幅比$AR = K_p \,/\, \sqrt{1 + \omega^2\tau^2}$
    相位滯後$\phi = -\arctan(\omega\tau)$
    趨勢$\uparrow\omega$ 或 $\uparrow\tau$ → $\downarrow AR$，相位滯後增加
    物理概念一階系統 = 低通濾波器

    範例 4.3（Coughanowr 教科書，第91頁）

    一支時間常數 $\tau = 0.1$ 分鐘的溫度計，初始溫度為 $25°$C，突然浸入 $40°$C 的介質中。

    求：

    1. (a) 0.1 分鐘後的溫度計讀數
    2. (b) 溫度計讀數達到 $36°$C 所需的時間

    詳細解題過程

    設定：

    已知：$\tau = 0.1$ 分鐘，$T_{m,s} = 25°$C（初始穩態），$T_s = 25°$C。

    新的周圍溫度：$T = 40°$C。

    輸入階躍變化：$A = T - T_s = 40 - 25 = 15°$C。

    溫度計的 $K_p = 1$。

    偏差變數響應：$y(t) = K_p A(1 - e^{-t/\tau}) = 15(1 - e^{-t/0.1})$

    實際溫度：$T_m(t) = T_{m,s} + y(t) = 25 + 15(1 - e^{-t/0.1})$

    (a) $t = 0.1$ 分鐘時的讀數

    由於 $t = 0.1 = \tau$：

    $$T_m(0.1) = 25 + 15(1 - e^{-0.1/0.1}) = 25 + 15(1 - e^{-1})$$ $$= 25 + 15(0.6321) = 25 + 9.48 = \boxed{34.48°\text{C}}$$

    在 $t = \tau$ 時，溫度計已達到總變化的 63.2%。

    (b) 讀數達到 $36°$C 的時間

    令 $T_m(t) = 36$：

    $$36 = 25 + 15(1 - e^{-t/0.1})$$ $$11 = 15(1 - e^{-t/0.1})$$ $$1 - e^{-t/0.1} = \frac{11}{15} = 0.7333$$ $$e^{-t/0.1} = 0.2667$$ $$-t/0.1 = \ln(0.2667) = -1.3218$$ $$\boxed{t = 0.1322 \text{ min} \approx 7.93 \text{ sec}}$$

    範例 4.3 — 正弦輸入（圖 4–18）

    同一支溫度計（$\tau = 0.1$ min）放入溫度波動的液體中：$T(t) = 100 + 2\sin(\omega t)$，週期 $P = 0.314$ min（$\omega = 2\pi/P \approx 20$ rad/min）。

    💡
    紅色實線 = 液體溫度：振幅 $= 2°$C。藍色虛線 = 溫度計讀數：$AR = 1/\sqrt{1+(20\times0.1)^2} = 1/\sqrt{5} \approx 0.447$，振幅 $= 2 \times 0.447 = 0.894°$C。相位滯後 $= 0.056$ min。溫度計無法跟上快速振盪！

    總結：所有一階響應

    對於 $G(s) = \dfrac{K_p}{\tau s + 1}$：

    輸入類型$x(t)$$X(s)$$Y(t)$
    階躍$A$$\dfrac{A}{s}$$K_pA(1-e^{-t/\tau})$
    脈衝$A\delta(t)$$A$$\dfrac{K_pA}{\tau}e^{-t/\tau}$
    斜坡$at$$\dfrac{a}{s^2}$$K_pa\!\left(t-\tau+\tau e^{-t/\tau}\right)$
    正弦$A\sin\omega t$$\dfrac{A\omega}{s^2+\omega^2}$$\dfrac{K_pA}{\sqrt{1+\omega^2\tau^2}}\sin(\omega t+\phi)$
    + 暫態

    總結：關鍵參數

    參數符號意義
    時間常數$\tau$階躍響應達到 63.2% 的時間；表徵響應速度
    穩態增益$K_p$穩態時輸出與輸入之比
    振幅比$AR$$K_p/\sqrt{1+\omega^2\tau^2}$；正弦輸入時輸出/輸入振幅比
    相位滯後$\phi$$-\arctan(\omega\tau)$；輸出滯後輸入的角度
    轉角頻率$\omega_c = 1/\tau$波德圖轉折點；$AR$ 下降 $3$ dB，$\phi = -45°$

    統整比較：增加 $\tau$ 的影響

    階梯、斜坡和正弦三種輸入，其實是從三個不同角度檢驗同一件事：一階系統有慣性，不可能瞬間、完美地跟上輸入。

    輸入類型核心問題增加 $\tau$ 的後果物理結論
    階梯多快到新穩態？到達新穩態更慢系統更遲鈍
    斜坡能否持續追上？追隨落後更明顯追隨能力變差
    正弦如何處理振盪？振幅更小、相位滯後更大更傾向濾除快速振盪
    💡
    章節總結：一階系統是程序控制中最基本的動態模型。對 step 輸入，可理解系統到達新穩態的速度（建立 $\tau$ 的概念）；對 ramp 輸入，分析系統對持續變動訊號的追隨能力；對 sinusoidal 輸入，描述系統對不同頻率訊號的振幅衰減與相位滯後。整體而言，$\tau$ 越大，表示系統越慢、越遲鈍。
